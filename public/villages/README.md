# Village Images

Place your village images here with filenames matching the `"image"` field in `public/data/villages.json`.

## Required images (based on sample data):
- `ngoai.png` — Thôn Ngoại
- `noi.png` — Thôn Nội
- `dinh.png` — Thôn Đình
- `chua.png` — Thôn Chùa
- `doai.png` — Thôn Đoài
- `bac.png` — Thôn Bắc
- `trung.png` — Thôn Trung
- `nam.png` — Thôn Nam

## Image Specifications
- Recommended: 1280×720 px or larger
- Format: PNG or JPEG
- If an image is missing, the app shows a styled placeholder automatically

## Notes
- Image filenames are case-sensitive on Linux/macOS servers
- Keep filenames simple (lowercase, no spaces, no special chars)
